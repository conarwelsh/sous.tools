# @sous/features

The shared business logic and component library for the `sous.tools` platform.

## Responsibilities

- **Domain Logic**: Reusable hooks and services shared between Web and Native.
- **Organisms**: Complex UI patterns (e.g., Recipe Editor, Inventory List).
- **Nested DDD**: Organized by Tactical Feature under Strategic Umbrellas.

## Functionality List

- [x] Universal logic core.
- [ ] Shared Server Actions for Next.js.
- [ ] Cross-platform React hooks.

## Tech Stack

- **React 19**: UI framework.
- **Socket.io Client**: Realtime hardware communication.
- **Lucide React**: Iconography.
- **Radix UI**: Accessible primitives.
- **React Markdown**: Documentation rendering.
