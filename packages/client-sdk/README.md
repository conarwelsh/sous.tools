# @sous/client-sdk

The generated SDK for all client-server communication in the `sous.tools` platform.

## Responsibilities

- **Type Safety**: Provides end-to-end typing for all API endpoints.
- **Abstraction**: Encapsulates REST, GraphQL, and WebSocket protocols.
- **Orchestration**: Simplifies complex multi-step API interactions.

## Functionality List

- [ ] Generated REST client.
- [ ] Generated GraphQL queries/mutations.
- [ ] Real-time Socket.io event types.
