# @sous/typescript-config

The shared TypeScript configuration for the `sous.tools` platform.

## Responsibilities
- **Standardization**: Provides base `tsconfig.json` files for different environments (Node, React, Next.js).
- **Paths**: Manages monorepo workspace aliases.
