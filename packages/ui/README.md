# @sous/ui

The universal UI component library for the `sous.tools` design system.

## Responsibilities
- **Atomic Components**: Button, Input, Text, View.
- **Design Tokens**: Standardized colors, spacing, and typography.
- **Brand Identity**: Logo variants and branding logic.

## Functionality List
- [x] React Native Web + NativeWind v4 setup.
- [x] Standardized brand logo registry.
- [ ] Accessible primitives via `@rn-primitives`.

## Usage
Uses Tailwind CSS classes compatible with both web and mobile environments.

## Tech Stack
- React Native
- NativeWind (Tailwind CSS)
- Lucide React Native
