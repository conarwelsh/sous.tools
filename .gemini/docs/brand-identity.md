# Brand Identity: sous.tools

## 1. Voice & Tone
- **Professional:** We are a mission-critical tool for chefs.
- **Efficient:** No fluff. Information must be digestible in high-heat, high-stress environments.
- **Reliable:** The UI should feel solid and unbreakable.

## 2. Visual Style (The "Sous" Look)
- **High Contrast:** Mandatory for visibility in bright/glare-heavy kitchens.
- **Functional Density:** Information-dense but well-spaced for large touch-screen interactions (Gloves support).
- **Industrial Minimal:** Clean lines, robust components, inspired by stainless steel and commercial kitchen aesthetics.

## 3. Core Color Palette (oklch)
- **Background:** `oklch(0.98 0.005 240)` (Light) / `oklch(0.12 0.01 240)` (Dark)
- **Foreground:** `oklch(0.12 0.02 240)` (Light) / `oklch(0.98 0.01 240)` (Dark)
- **Primary (Brand):** `oklch(0.60 0.25 250)` (Azure Blue). Used for Production.
- **Success:** `oklch(0.70 0.25 150)` (Emerald Green). Used for Development.
- **Warning:** `oklch(0.85 0.20 85)` (Vivid Amber). Used for Staging.
- **Accent:** `oklch(0.65 0.25 45)` (Solar Orange).
- **Destructive:** `oklch(0.60 0.25 25)` (Pure Crimson).

## 4. Typography
- **Headings:** Sans-serif, bold, tight tracking.
- **Monospace:** Used for all "Numerical" culinary data (Weights, Temperatures, Timers) to ensure perfect alignment in lists.

## 5. Components & Atoms
- **Corner Radius:** `0.8rem` (Modern, professional rounding).
- **Borders:** `1px` solid.
- **Icons:** Lucide icons, consistent stroke width of `2px`.
