# @sous/native-kds

The Kitchen Display System (KDS) application for the `sous.tools` platform.

## Responsibilities
- **Order Management**: Real-time grid of incoming orders and preparation states.
- **Expediter View**: Centralized control for front-of-house coordination.
- **HACCP Logging**: Display and log real-time temperature telemetry.

## Functionality List
- [ ] Real-time order synchronization.
- [ ] Acoustic alerts for new tickets.
- [ ] Multi-screen station routing.

## Installation & Setup
1. Designed for Android-based 21.5" touch terminals.
2. Follow standard Tauri Android installation procedures.

## Tech Stack
- React
- Tauri v2
- `@sous/ui`
