# @sous/web

Primary Next.js web application for the SOUS platform.

## Features
- Server-Side Rendering (SSR) and Server Components.
- Universal UI components via `@sous/ui`.
- Subdomain-based multi-tenancy.

## Development
- Port: `3000`
- Command: `pnpm run dev`

## Tech Stack
- Next.js 16
- React
- Tailwind CSS / NativeWind
- TypeScript
