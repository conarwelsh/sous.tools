# @sous/native

Universal native application for the SOUS platform (iOS/Android/Desktop).

## Features
- Shared business logic with @sous/web.
- Native capabilities via Tauri 2.0.
- Offline-first caching (planned).

## Development
- Vite Port: `1421`
- Command: `pnpm run dev`
- Android Dev: `pnpm run android:dev` (requires connected device/emulator)

## Tech Stack
- React
- Tauri 2.0
- Rust (Native Bridge)
- TypeScript
