# @sous/api

Centralized NestJS API for the SOUS platform.

## Features
- Domain-Driven Design (DDD) architecture.
- Centralized configuration via `@sous/config`.
- Centralized logging via `@sous/logger`.
- REST API with Swagger documentation (planned).

## Development
- Port: `4000`
- Command: `pnpm run dev`

## Tech Stack
- NestJS
- Drizzle ORM
- TypeScript
